# 🚀 关于我

你好！我是一名充满热情的创客教育者，曾经在高压力的金融交易世界工作，现在找到了真正热爱的事业——**培养下一代创新者和创造者**。

## 📖 我的故事

### 从数字到创造
三年前，我还是一名金融交易员，每天面对着冰冷的数字屏幕。虽然收入可观，但我总觉得缺少些什么。一次偶然的机会，我参加了学校的创客教育活动，看到孩子们通过编程控制机器人时眼中闪烁的光芒，那一刻，我找到了人生的新方向。

### 转型的收获
- **发现了创造的乐趣**: 从交易数字变成了创造真实的产品
- **找到了教育的意义**: 看到学生成长比账户盈利更有成就感
- **建立了新的社群**: 认识了很多志同道合的创客朋友
- **重新定义了成功**: 培养创新思维比追求个人收益更有价值

## 🎯 使命与愿景

### 我的使命
通过**项目式学习**和**动手实践**，帮助每个孩子发现自己的创造潜能，培养21世纪需要的核心技能：批判性思维、创造力、协作能力和沟通能力。

## 🎨 教学理念

### 四大原则

1. **项目驱动**
   - 以实际问题为出发点
   - 在解决中学习知识和技能
   - 培养系统思维能力

2. **个性化指导**
   - 尊重每个学生的独特性
   - 根据兴趣和水平调整教学
   - 鼓励个性化创作

3. **跨学科融合**
   - 融合科学、技术、工程、艺术、数学
   - 培养综合解决问题的能力
   - 连接理论知识与实际应用

4. **协作共创**
   - 鼓励团队合作
   - 培养沟通能力
   - 建立学习社群

### 教学方法
- **做中学**: 通过动手实践掌握概念
- **游戏化**: 用游戏激发学习兴趣
- **迭代优化**: 不断改进和完善作品
- **分享展示**: 培养表达能力
