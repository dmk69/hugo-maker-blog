## 使用说明

这些图片用于展示后端开发技能和项目经验，可以在技能介绍、技术博客和简历材料中使用。

## 技术规格

- **格式**: WebP (主格式), PNG (备选格式)
- **尺寸**: 根据用途调整，图标一般200x200px，架构图宽度不超过1000px
- **压缩**: 优化展示效果，确保清晰度和文件大小平衡

## 注意事项

1. 使用官方或高质量的图标
2. 架构图应清晰易懂
3. 避免包含敏感的系统信息
4. 考虑添加简短说明或标注
