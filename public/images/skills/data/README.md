## 使用说明

这些图片用于展示数据分析技能和项目经验，可以在技能介绍、技术博客和简历材料中使用。

## 技术规格

- **格式**: WebP (主格式), PNG (备选格式)
- **尺寸**: 根据用途调整，图标一般200x200px，图表宽度不超过800px
- **压缩**: 优化数据可视化展示，确保图表清晰可读

## 注意事项

1. 使用官方或高质量的图标
2. 数据可视化应清晰易懂
3. 避免包含敏感的业务数据
4. 考虑添加简短说明或标注
5. 确保图表标签和图例清晰可读
