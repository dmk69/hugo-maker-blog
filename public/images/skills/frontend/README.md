## 技术规格

- **格式**: WebP (主格式), PNG (备选格式)
- **尺寸**: 根据用途调整，图标一般200x200px，截图宽度不超过800px
- **压缩**: 优化展示效果，确保清晰度和文件大小平衡

## 注意事项

1. 使用官方或高质量的图标
2. 代码示例应清晰可读
3. 项目截图应展示真实效果
4. 考虑添加简短说明或标注
