## 技术规格

- **格式**: WebP (主格式), PNG (备选格式)
- **尺寸**: 桌面端截图，一般宽度不超过1200px
- **压缩**: 优化界面展示，确保文字和图标清晰

## 注意事项

1. 图片应展示真实的工具界面
2. 确保代码示例和配置清晰可读
3. 避免包含敏感的API端点或密钥
4. 考虑添加高亮和标注以突出重要功能
