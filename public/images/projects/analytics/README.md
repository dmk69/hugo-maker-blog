## 使用说明

这些图片用于展示销售数据分析仪表板的功能和设计，可以在项目介绍、技术博客和演示材料中使用。

## 技术规格

- **格式**: WebP (主格式), PNG (备选格式)
- **尺寸**: 桌面端截图，一般宽度不超过1200px
- **压缩**: 优化数据可视化展示，确保图表清晰可读

## 注意事项

1. 图片应展示真实的数据可视化效果
2. 确保图表标签和数据清晰可读
3. 避免包含敏感业务数据
4. 考虑添加交互效果说明
