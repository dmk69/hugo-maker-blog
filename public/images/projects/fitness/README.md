## 技术规格

- **格式**: WebP (主格式), PNG (备选格式)
- **尺寸**: 移动端截图，一般宽度不超过400px
- **压缩**: 优化移动端显示，确保快速加载

## 注意事项

1. 图片应展示真实的移动应用界面
2. 确保在不同设备上的显示效果
3. 避免包含个人健康数据
4. 考虑添加设备框架以增强展示效果
