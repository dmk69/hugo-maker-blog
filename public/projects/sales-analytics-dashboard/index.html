销售数据分析仪表板#

这是一个基于Python的销售数据分析仪表板，为企业提供实时数据可视化和预测分析功能。

项目概述#

销售数据分析仪表板是一个综合性的商业智能解决方案，帮助企业分析销售数据、识别趋势、预测未来销售情况，并提供数据驱动的决策支持。系统集成了数据收集、清洗、分析和可视化功能，采用现代化的数据科学技术栈。

class SalesPredictor:
    def __init__(self):
        self.models = {
            'linear_regression': LinearRegression(),
            'random_forest': RandomForestRegressor(),
            'xgboost': XGBRegressor(),
            'lstm': self._build_lstm_model()
        }
    
    def train(self, X_train, y_train):
        """训练多个预测模型"""
        results = {}
        for name, model in self.models.items():
            model.fit(X_train, y_train)
            predictions = model.predict(X_test)
            results[name] = {
                'mae': mean_absolute_error(y_test, predictions),
                'rmse': np.sqrt(mean_squared_error(y_test, predictions)),
                'r2': r2_score(y_test, predictions)
            }
        return results
    
    def predict(self, X, model_name='ensemble'):
        """使用集成模型进行预测"""
        if model_name == 'ensemble':
            predictions = []
            for model in self.models.values():
                pred = model.predict(X)
                predictions.append(pred)
            return np.mean(predictions, axis=0)
        else:
            return self.models[model_name].predict(X)

class AnomalyDetector:
    def __init__(self, method='isolation_forest'):
        self.method = method
        if method == 'isolation_forest':
            self.model = IsolationForest(contamination=0.1)
        elif method == 'one_class_svm':
            self.model = OneClassSVM(nu=0.1)
    
    def detect_anomalies(self, data):
        """检测数据中的异常点"""
        anomalies = self.model.fit_predict(data)
        return anomalies == -1  # 返回异常点的布尔数组

# docker-compose.yml
version: '3.8'
services:
  api:
    build: ./api
    ports:
      - "8000:8000"
    environment:
      - DATABASE_URL=postgresql://user:pass@db:5432/sales_db
    depends_on:
      - db
      - redis
  
  frontend:
    build: ./frontend
    ports:
      - "3000:3000"
  
  db:
    image: postgres:13
    environment:
      - POSTGRES_DB=sales_db
      - POSTGRES_USER=user
      - POSTGRES_PASSWORD=pass
    volumes:
      - postgres_data:/var/lib/postgresql/data
  
  redis:
    image: redis:6-alpine
    ports:
      - "6379:6379"
